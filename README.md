# firmas_corporativas

Recursos y plantillas para las firmas de correo corporativas del Grupo GALA (GALA Formación y Autoescuela GALA).

## Estructura

```
firmas_corporativas/
├── logos/        Logos de marca (PNG)
├── iconos/       Iconos de redes sociales (50px)
├── plantillas/   4 plantillas HTML con placeholders {{...}}
│   ├── formacion-con-qr.html
│   ├── formacion-sin-qr.html
│   ├── autoescuela-con-qr.html
│   └── autoescuela-sin-qr.html
└── qr/           QRs personales por empleado (qr-{slug}.png)
```

Los assets se sirven vía GitHub Pages en `https://grupo-gala.github.io/firmas_corporativas/`.
